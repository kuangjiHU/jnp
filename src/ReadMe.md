## 第二章 流

JAVA的IO建立在流之上。
虽然不同的流会读写某个特顶的数据源，但是所有的流都有相同的基本方法来写入、读取数据。

- 过滤器流，读写数据时，可以修改数据，或者只是提供额外的方法，讲述转换为其他的格式。
- 阅读器、书写器允许读写文本（及字符）而不是字节。可以处理很多字符编码。
- 流是同步的，所以在线程请求一个流读写一段数据是，在做任何其他操作前，它要等待所读写的数据。
- java还支持使用通道和缓冲区的非阻塞IO.
